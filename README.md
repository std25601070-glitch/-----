# NURAI — Medical Assistant | المساعد الطبي الذكي

> AI-powered medical triage system for initial health assessments  
> نظام ذكاء اصطناعي للتقييم الصحي الأولي

🔗 **Live Demo:** [nurai-medical-assistant.vercel.app](https://nurai-medical-assistant.vercel.app)

---

## Overview

NURAI is a web-based medical triage assistant that guides users through 
a structured health assessment and produces an AI-powered initial 
evaluation — without replacing a doctor.

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | Next.js 16 + React 19 |
| Language | TypeScript |
| Styling | Tailwind CSS v4 |
| UI Components | Shadcn/UI + Radix UI |
| Animation | Framer Motion |
| Forms | React Hook Form + Zod |
| Deployment | Vercel |
